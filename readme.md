# Express ts starter

## Requirement
##### MongoDB
Install the server version of[MongoDB](https://www.mongodb.com/download-center/community) or use a docker image.
##### Config file
Edit the config or create .env file in root folder
```
NODE_ENV=development
PORT=3002
DEBUG=app:*
```

## Install
1. `yarn install`
2. `yarn run init`

## Launch
`yarn run dev`
